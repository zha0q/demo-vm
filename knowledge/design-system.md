# 设计系统知识

## 主要参考

以 `.agent/Design.md` 作为详细视觉规范。本文件是给后续实现使用的简版工作摘要。

## 视觉方向

整体风格是：

- 东方感。
- 首页使用温暖木色。
- 游戏页使用深翡翠绿色毛毡。
- 完成页使用金色仪式感覆盖层。
- 圆润、光亮、触感强的移动游戏控件。
- 带绿色边缘的 3D 象牙色麻将牌。

不要复制 Vita Mahjong 的 Logo、原始美术、按钮纹理、牌面绘图、音效或品牌文案。

## 页面主题

| 页面 | 视觉主题 |
|---|---|
| 首页 | 木门、宣纸窗、竹影、橙金色开始按钮 |
| 游戏页 | 深绿色毛毡桌面、木质圆形按钮、四格深色队列、居中的 3D 牌面棋盘 |
| 完成页 | 深绿色覆盖层、金色光晕、莲花/云纹装饰、统计卡片、绿色下一步按钮 |

## 核心令牌

使用 `.agent/Design.md` 中的这些令牌家族：

- 翡翠绿：`--color-jade-*`
- 木色棕：`--color-wood-*`
- 金色：`--color-gold-*`
- 奶油色文字/牌面：`--color-cream-100`、`--color-tile-face`
- 牌边绿色：`--color-tile-edge`
- 红色徽标：`--color-red-600`

## 布局规则

- 根节点使用 `100vw` 和 `100dvh`。
- 页面不能滚动。
- 文字和控件使用固定像素尺寸。
- 间距和 3D 摄像机取景要随视口自适应。
- 顶部 IQ 和队列固定在靠上位置。
- 底部操作按钮固定在安全区域底部附近。
- 棋盘占据剩余的中央区域。

目标视口：

- 375x667
- 390x844
- 430x932
- 768x1024

## 组件说明

首页：

- 左上角头像框。
- 右上角圆形木质图标按钮。
- 上半区的大标题。
- 底部橙金色胶囊按钮。

游戏页：

- 圆形木质返回按钮。
- 中间显示“清台 IQ”、数值和标签。
- 圆形木质更多按钮。
- IQ 下方是四格队列。
- 中央是 3D 棋盘。
- 底部有三个圆形操作按钮，并带红色数量徽标。
- 更多菜单从右侧主按钮向左展开，当前包含音效、洗牌和重开入口。

完成页：

- 深色覆盖在游戏画面上。
- 金色标题和莲花/云纹主题。
- 三张统计卡片：时间、IQ、连击。
- 奖励文案里把关键数字高亮为绿色。
- 进度条朝向奖励宝箱。
- 绿色胶囊形下一步按钮。

## 动效

- 首页标题：柔和下落并缩放。
- 首页按钮：轻微高亮脉冲。
- 选牌：抬升并发出边缘光。
- 无效点击：轻微抖动。
- 队列配对：滑到一起、碰撞、破碎/掉落。
- 提示：1.2 秒脉冲。
- 完成：深色淡出、金色光晕、卡片依次出现。

## 质检

完成视觉工作前，请检查：

- 没有滚动条。
- 没有重叠。
- 文字在控件内部显示完整。
- 画布不是空白。
- 棋盘没有被底部按钮挡住。
- 红色徽标清晰可读。
